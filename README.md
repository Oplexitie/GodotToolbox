# GodotToolbox

A collection of scripts I use across my Godot projects.

#### Autoload

- Event Bus 
- Scene Manager
- Two Audio Managers (AudioChannels and AudioPool)
- Save Manager
- Settings Manager 

#### Other

- State Machine
- Animation State Manager

## Other info
